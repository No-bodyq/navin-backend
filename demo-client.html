1. Send Telemetry Data

{

  "humidity": 55,
  "latitude": 12.34,

2. Update Shipment Status

PATCH http://localhost:3000/api/shipments/YOUR_SHIPMENT_ID/status
Headers:
  Authorization: Bearer YOUR_JWT_TOKEN
  Content-Type: application/json
Body:
{
  "status": "IN_TRANSIT"
}
